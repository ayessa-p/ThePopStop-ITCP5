Setup Guide: Running The Pop Stop on a New Device

Follow these steps to get the system running after downloading the project folder.

1. Prerequisites

Before starting, ensure the new device has the following installed:

  • PHP (8.2 or higher)
  • Composer (Dependency manager for PHP)
  • MySQL (or a local server like XAMPP/WAMP)
  • Node.js & NPM (For frontend assets)

2. Step-by-Step Installation

Step 1: Open the Project Folder

Open your terminal (Command Prompt, PowerShell, or VS Code Terminal) and navigate to the project folder:

cd ThePopStop

Step 2: Install PHP Dependencies

Run this to install all the necessary Laravel libraries:

composer install

Step 3: Setup Environment File

Create your local environment file by copying the example:

cp .env.example .env

Step 4: Generate Application Key

This is required for Laravel's security and encryption:

php artisan key:generate

Step 5: Configure the Database

  1. Open your MySQL manager (like phpMyAdmin).
  2. Create a new database named thepopstop.
  3. Open the .env file in the project folder and update the database section:
DB_DATABASE=thepopstop
DB_USERNAME=root
DB_PASSWORD=

Step 6: Run Migrations and Seeders

This will create all the tables and the default Admin account:

php artisan migrate --seed

Step 7: Install and Build Frontend Assets

Install the JavaScript libraries and build the CSS/JS files:

npm install
npm run build

Step 8: Link Storage

This ensures that uploaded photos (products and profiles) are accessible:

php artisan storage:link

3. Running the System

Start the local development server:

php artisan serve

The system will be accessible at: http://127.0.0.1:8000
