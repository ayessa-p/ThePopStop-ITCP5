Setup Guide for The Pop Stop

Follow these steps to set up and run the project on your new device.

Prerequisites

  • PHP (>= 8.2)
  • Composer
  • Node.js & NPM
  • MySQL/MariaDB (XAMPP or WAMP recommended)

Setup Instructions

1. Database Configuration

  1. Open your database manager (e.g., phpMyAdmin).
  2. Create a new database named thepopstop.

2. Environment Setup

  1. Open the project folder on your new device.
  2. Copy .env.example and rename it to .env.
  3. Update the database settings in .env:
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=thepopstop
DB_USERNAME=root
DB_PASSWORD=

3. Install Dependencies

Open a terminal inside the project folder and run:

composer install
npm install

4. Application Key

php artisan key:generate

5. Migration and Seeding

This will recreate all tables and restore all data:

php artisan migrate:refresh --seed

6. Search Index

php artisan scout:import "App\Models\Product"

7. Link Storage

php artisan storage:link

8. Compile Assets

npm run build

Running the Application

To start the server, run:

php artisan serve

Then visit: http://127.0.0.1:8000
